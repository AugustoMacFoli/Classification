K-Nearest Neighbors (KNN)

KNN Algorithm is based on feature similarity: How closely out-of-sample features resemble our training set determines how we classify a given data point.

KNN could and probably should be one of the first choices for a classification study when there is little or no prior knowledge about the distribution data. KNN can be used for both classification and regression predictive problems. However, it is more widely used in classification problems. It usually requires more computing power than other algorithms.

This example will try to predict if a client will or will not buy a product based on the client's age and estimated income.
